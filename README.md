# 💰 Expense-Tracker  
## SpendSense – AI Expense Tracker

SpendSense is a modern **AI-powered expense tracking web application** that helps users monitor spending, manage budgets, and gain intelligent financial insights.

---

## 🚀 Features

### 📊 Dashboard Overview
- View total monthly spending, remaining budget, and transaction count  
- Highlights top spending category  
- Clean and intuitive financial snapshot  

---

### 🧾 Expense Management
- Add, categorize, and track daily expenses  
- Categories include:  
  - 🍔 Food  
  - 🚗 Transport  
  - 🛍️ Shopping  
  - 🏥 Health  
  - 🎬 Entertainment  
  - ⚡ Utilities  
  - 📦 Others  

---

### 📈 Budget Tracking
- Set monthly budgets per category  
- Visual progress bars for budget usage  
- Alerts when nearing or exceeding limits  

---

### 📉 Analytics & Visualization
- Interactive daily spending graph (last 30 days)  
- Category-wise expense breakdown  

---

### 🤖 AI-Powered Insights
- Generates personalized financial tips  
- Highlights overspending patterns  
- Provides actionable suggestions with ₹ amounts  

---

### 💬 AI Financial Advisor
- Chat-based assistant  
- Ask questions like:  
  - *How can I reduce my expenses?*  
  - *Where am I overspending?*  
- Get quick, smart, and contextual answers  

---

### 🔮 Expense Forecasting
- Predicts spending for the next 3 months  
- Shows trends:  
  - ⬆️ Increasing  
  - ➡️ Stable  
  - ⬇️ Decreasing  
- Includes reasoning behind predictions  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Charts:** Chart.js  
- **AI Integration:** Google Gemini API  
- **Storage:** LocalStorage
